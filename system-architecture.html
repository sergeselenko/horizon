Slash CommandsText-based workflow triggers — Claude reads a Notion prompt page and follows the procedure
/closeout▸
Close session. Session Log, Continuity Check.
When: End of any substantive session
Produces: Session Log entry, Chat Map update, snapshot/digest check
/tasks▸
Scan conversation for tasks to Notion.
When: After /closeout
Rule: Commands never auto-chain
/capture▸
Harvest instruction improvements + decision records.
When: After /tasks — the learning loop
Searches for: /flag markers from mid-session
/emit▸
Cross-project state propagation.
Writes to: Global Operator Profile + Secretary Pad
Optional: Venture Posture Table updates
/digest▸
Update Knowledge Digest with new domain intel.
When: Competitive data, pricing, people observations surface
/snapshot▸
Mid-session Project Snapshot update.
Scoped to: Project Snapshot page only
/flag▸
Mid-session course correction bookmark.
Example: /flag — ran 5 searches before reading the obvious page
/scope▸
Lock session scope. Claude flags drift.
Expand: “Add [X] to scope” at any time
